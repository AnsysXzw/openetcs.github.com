Get in touch with openETCS

There are various ways to get in touch with the openETCS project, depending on the matter.

  • openETCS newsgroup
    Please use this newsgroup for general questions regarding openETCS
  • openETCS project oewsgroups (Project Listing)
    If you have a question regarding a specific project lease use the project usegroup. You find the link on each of the project pages.
  • openETCS Mentoring Board
    If you are interested in participating or if you have a question that you don't want to discuss publicly you can get in touch with the openETCS mentoring board.
