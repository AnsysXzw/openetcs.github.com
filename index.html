European Train Control System (ETCS)
Open Proof. Open Source.

The European Union requires to replace all legacy signalling and train control systems with the new European Train Control System. The ETCS “System Requirement Specification” is public domain. This opens up the opportunity to apply the “open proofs” methodology, by providing formal system requirement and use case specification, embedded control software, interfaces, architecture, tools and safety case documents all under an open source license.
