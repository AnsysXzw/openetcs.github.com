Components

Currently, openETCS consists of the following Components:

  • Eclipse Kepler
  • Eclipse Modeling Tools
  • Eclipse Papyrus
  • Eclipse RMF
  • Eclipse EGit
  • openETCS DataDictionary (under Construction)
