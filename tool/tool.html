Components

Currently, openETCS consists of the following Components:

  • Eclipse Kepler
  • Eclipse Modeling Tools
  • Eclipse Papyurus
  • Eclipse RMF
  • Eclipse EGit
  • openETCS DataDictionary (under Construction)

Current Status

We are in the process of setting up the development processes and the project infrastructure. The first release is a set of installation instructions. Work on the Data Dictionary Component started.
